# Dataset Source and Redistribution Notes

## BIRDS EPS Data Source Record

This reproducibility package references processed artifacts derived from BIRDS CubeSat EPS telemetry data used in the T5 safety-gated compression study.

## Raw Data Inclusion Status

Raw telemetry data is not included in this public package. Only final derived result tables and summaries needed for paper reproducibility are included.

## Checksum and Source Information

- Reproducibility checksums and manifest are provided in:
  - `results/repro/t5_checksums.txt`
  - `results/repro/t5_repro_manifest.json`
- Environment snapshot is provided in:
  - `results/repro/t5_environment.txt`

## Redistribution Caution

If upstream dataset licensing terms are unclear, do not redistribute raw source telemetry. Dataset assets retain their original licenses and usage constraints from their original providers.
